# /nvChatTest - 角色對話測試

模擬多角色對話，確認語氣區分和一致性。

## 參數

支援角色名或角色 ID，可以指定 2-4 個角色：

| 參數 | 必填 | 說明 | 範例 |
|------|------|------|------|
| `proj` | ✅ | 專案名稱 | `proj=霓虹劍仙` |
| 角色 | ✅ | 角色名或ID (2-4個) | `"李玄" 1 144` |
| `topic` | ❌ | 對話主題 | `topic=交易` |
| `mood` | ❌ | 情緒基調 | `mood=hostile` |

## 使用範例

```
/nvChatTest proj=霓虹劍仙 "李玄" "王執法"
/nvChatTest proj=霓虹劍仙 1 2 3 topic=交易
/nvChatTest proj=劍來 "主角" 5 mood=friendly
```

## 執行步驟

### Step 1: 載入角色
// turbo
從 `character_db.yaml` 讀取指定角色的 `base_profile`：
- `speech_pattern`
- `traits`
- 派系背景

### Step 2: 設定場景
根據 `topic` 和 `mood` 設定對話情境

### Step 3: 生成對話
使用 `skill_dialogue_director` 生成測試對話：
- 5-10 個回合
- 每個角色至少發言 2 次

### Step 4: 分析結果
輸出：
```
## 對話測試結果

### 角色語氣區分
| 角色 | 特徵 | 範例句 |
|------|------|--------|
| 李玄 | 寡言、技術術語 | 「帶寬不足。撤。」 |
| 王執法 | 官腔、威脅語氣 | 「依據第三十七條...」 |

### 潛台詞分析
- 李玄：表面配合，實際在找逃跑路線
- 王執法：表面執法，實際在試探情報

### 一致性評估
- ✅ 李玄語氣符合設定
- ⚠️ 王執法用詞過於輕鬆，略有偏差

### 建議調整
- 王執法可增加更多法律術語
```

## 輸出

對話測試結果到終端，可用於微調角色設定。
