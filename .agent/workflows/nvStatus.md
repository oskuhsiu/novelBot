# /nvStatus - 狀態總覽

顯示專案的當前狀態與進度。

## 參數

| 參數 | 必填 | 說明 | 範例 |
|------|------|------|------|
| `proj` | ✅ | 專案名稱 | `proj=霓虹劍仙` |

## 使用範例

```
/nvStatus proj=霓虹劍仙
```

## 執行步驟

### Step 1: 讀取進度
// turbo
讀取 `narrative_progress.yaml`

### Step 2: 生成報告
// turbo
輸出：

```
## 專案：{name}

### 進度
- 當前章節：{current_chapter} / {total_chapters}
- 完成比例：{percent}%
- 總字數：{words_written}

### 角色狀態
| 角色 | 位置 | 健康 | 情緒 |
|------|------|------|------|
| 李玄 | 聽劍閣 | 95% | 警覺 |
| ... | ... | ... | ... |

### 伏筆追蹤
- 開放伏筆：{open_count}
- 休眠警告：{dormant_list}
- 即將揭曉：{ready_list}

### 勢力緊張度
| 勢力A | 勢力B | 緊張度 |
|-------|-------|--------|
| 天宮 | 叛軍 | 95/100 |

### 最近事件
- Ch.5: 主角擊敗王執法
- Ch.4: 獲得加密硬盤
```

## 輸出

狀態報告到終端。
