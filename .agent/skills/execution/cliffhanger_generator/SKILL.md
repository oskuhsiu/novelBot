# 懸念製造機 (Cliffhanger Generator)

## 功能概述

此 Skill 專門設計章節結尾的懸念，確保讀者有繼續閱讀的慾望。好的懸念是「讀完這章必須馬上看下一章」的關鍵。

## 懸念類型

| 類型 | 情緒 | 範例 |
|------|------|------|
| 危險逼近型 | 恐懼、緊張 | 門被踢開，黑影出現在門口 |
| 秘密揭曉型 | 震驚、好奇 | 「你不知道嗎？他是你父親」 |
| 兩難選擇型 | 焦慮、期待 | 左邊是愛人，右邊是使命 |
| 反轉型 | 錯愕、興奮 | 看似失敗，實則是計畫的一部分 |
| 懸疑深化型 | 好奇、困惑 | 發現新線索，引出更大謎團 |
| 情感衝擊型 | 心痛、期待 | 告白後章節結束 |
| 突發終止型 | 驚恐、不安 | 戰鬥進行到一半戛然而止 |

## 輸入

1. 本章內容摘要
2. 下一章預計發展
3. `pacing_pointer`（決定懸念強度）

## 輸出

1. 懸念方案（3 選 1）
2. 具體結尾句建議
3. 下一章開頭銜接建議

## 執行步驟

### Step 1: 分析本章定位
```
讀取章節資訊：
- 章節結構位置：{{structure_role}}
- 本章情緒曲線：{{emotion_arc}}
- 本章高潮點：{{chapter_climax}}
- 下一章預計發展：{{next_chapter_summary}}
```

### Step 2: 選擇懸念策略
```
## 懸念強度與 pacing_pointer

【低 pacing (0.1-0.3)】
- 懸念可以較輕
- 著重情感餘韻
- 可以是疑問而非危機

【中 pacing (0.4-0.6)】
- 標準懸念強度
- 問題提出，解答待續

【高 pacing (0.7-1.0)】
- 強烈懸念
- 戰鬥中斷、高潮中止
- 強迫效果

## 結構位置考量

- 第一幕：懸念引導向更大的世界
- 中點：反轉型懸念最有效
- 第三幕前：危險逼近型效果最好
- 結局前：所有線索即將匯合的暗示
```

### Step 3: 生成懸念方案
```
請為本章結尾設計 3 個不同類型的懸念：

【本章內容】：{{chapter_summary}}
【下一章走向】：{{next_chapter_direction}}
【可用元素】：{{available_elements}}

## 方案 A：{{type_A}} 型

結尾場景：
{{scene_description}}

結尾句建議：
「{{closing_line}}」

下一章銜接：
{{next_chapter_hook}}

## 方案 B：{{type_B}} 型
...

## 方案 C：{{type_C}} 型
...

請選擇最符合故事需求的方案。
```

## 懸念技法

### 技法 1：截斷
在最緊張的時刻戛然而止：
```
「刀鋒距離他的喉嚨只有三寸——」
章節結束。
```

### 技法 2：延遲揭示
給出線索但不給答案：
```
她認出了那個標記。
如果這是真的，那麼一切都說得通了。
她明白了為什麼父親一直要隱瞞真相。
```

### 技法 3：意外轉折
在讀者以為塵埃落定時：
```
他終於鬆了一口氣，危機解除了。
然後他看到了天空——
那裡有什麼東西正在墜落。
```

### 技法 4：問題升級
解決一個問題，揭示更大的問題：
```
他們找到了鑰匙。
但打開門後，看到的不是他們預期的寶藏——
而是一整支軍隊。
```

### 技法 5：情感炸彈
在情感最高漲時結束：
```
「我……」她第一次沒有躲開他的目光。
他等待著那個詞語。
這個等待，持續到了下一章。
```

## 懸念範例庫

### 動作類
```
- 「劍刃停在他的眉心，他看到了倒映的自己——正在笑。」
- 「爆炸倒計時顯示還剩3秒。出口在50米外。」
- 「增援終於到了。只是，來的不是他們期待的人。」
```

### 懸疑類
```
- 「答案就在眼前，但他不確定自己想不想知道。」
- 「那個死了三年的人，正站在他的面前。」
- 「調查報告的最後一行寫著一個名字——他的名字。」
```

### 情感類
```
- 「他轉身離開，沒有回頭。她沒有叫住他。」
- 「這三個字，他等了十年才說出口。然後，沉默。」
- 「她握著那封信，不知道是該燒掉，還是讀完它。」
```

## 使用時機

- **章節結尾**：每章最後一個節拍
- **中場休息**：較長章節的中間也可以設置小懸念

## 與其他 Skill 的關聯

- **前置**：`skill_scene_writer`（需要知道本章內容）
- **參考**：`skill_outline_architect`（了解整體結構）

## 注意事項

1. **兌現承諾**：懸念製造的期待，下一章必須回應
2. **不要每次都用相同技法**：讀者會適應
3. **切勿過度**：不是每個懸念都要生死攸關
4. **類型匹配**：輕鬆小說的懸念和嚴肅小說不同
5. **自然流動**：懸念應該是劇情發展的自然結果，不是刻意安排
