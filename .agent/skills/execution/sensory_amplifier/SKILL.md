# 感官放大鏡 (Sensory Amplifier)

## 功能概述

此 Skill 針對描寫平淡的段落進行潤色，加入具體的五感細節：
- 視覺 (Visual)
- 聽覺 (Auditory)
- 嗅覺 (Olfactory)
- 觸覺 (Tactile)
- 味覺 (Gustatory)

## 核心原則

1. **具體勝於抽象**：「聞起來像工業油」比「有味道」好
2. **相關性**：只描寫對場景有意義的感官
3. **角色化**：不同角色會注意不同的細節
4. **情緒聯結**：感官應該強化情緒氛圍

## 輸入

1. 需要潤色的段落
2. 場景類型和氛圍
3. 視角角色

## 輸出

強化後的段落

```
【嗅覺 (Olfactory)】
- 主要氣味：{{最明顯的氣味}}
- 次要氣味：{{隱藏的、微妙的}}
- 記憶連結：{{這個氣味讓角色想起什麼}}

【觸覺 (Tactile)】
- 溫度：{{冷熱}}

- 體感：{{疼痛、疲勞、舒適}}

【味覺 (Gustatory)】
- 口中餘味：{{剛吃過什麼}}
- 空氣味道：{{嘗得到的氣味}}
- 生理反應：{{口乾、噁心}}
```

### Step 3: 執行強化
```
## 強化策略

【避免過度】
- 不需要每段都有五感
- 選擇 2-3 個最相關的感官
- 感官描寫要為劇情服務

【融入敘事】
- 感官應該自然穿插，不是單獨成段
- 用動作帶出感官（「他揉了揉剛碰過金屬欄杆的手指，冰涼還沒散去」）
- 感官可以暗示時間流逝

【角色視角】
- 黑客可能會先注意電子設備的嗡響
- 廚師可能會先注意食物氣味
- 戰士可能會先評估環境的戰術價值

請重寫以下段落，融入感官細節：
{{original_text}}
```

## 感官範例庫

### 科幻/賽博龐克場景
```yaml
visual:
  - "霓虹燈管閃爍著不穩定的頻率"
  - "全息廣告在半空中糾纏重疊"
  - "雨水沿著污漬的玻璃幕牆流下，扭曲了城市的輪廓"

auditory:
  - "懸浮車引擎的低頻震動從頭頂掠過"
  - "遠處傳來警笛的多普勒效應"
  - "電弧焊接的嘶嘶聲混在人聲中"

olfactory:
  - "空氣過濾器遮掩了一半的工業廢氣味"
  - "某種化學甜味——可能是劣質的情緒補劑"
  - "燒焦的電線絕緣層味道"

tactile:
  - "潮濕的空氣在皮膚上留下一層薄薄的油膩"
  - "義體接口處傳來微弱的電流刺痛"
  - "防護衣內側已經被汗水浸透"
```

### 玄幻/修仙場景
```yaml
visual:
  - "靈氣在空中凝成淡淡的霧靄"
  - "劍光劃破黑暗時帶起一串星火"
  - "丹爐口氤氳著五彩斑斕的光暈"

auditory:
  - "遠處傳來修士御劍破空的尖嘯"
  - "丹田中真氣運轉的轟鳴只有自己能聽見"
  - "風穿過古鬆，帶著千年不變的低吟"

olfactory:
  - "藥材的苦香混著焦土的味道"
  - "血腥味蓋過了早晨的露水清新"
  - "禁地中彌漫著讓人頭暈的腐朽氣息"

tactile:
  - "靈力流經經脈時帶來的微麻"
  - "寒鐵劍柄貼著掌心，汲取體溫"
  - "傷口處傳來灼燒般的疼痛"
```

## 氛圍感官對照

| 氛圍 | 主要感官 | 建議細節 |
|------|----------|----------|
| 緊張 | 聽覺、觸覺 | 心跳、呼吸、冷汗 |
| 恐怖 | 視覺、聽覺 | 陰影、寂靜、突發聲響 |
| 溫馨 | 觸覺、嗅覺 | 溫暖、食物香氣 |
| 戰鬥 | 視覺、觸覺 | 動作、衝擊、疼痛 |
| 神祕 | 嗅覺、視覺 | 陌生氣味、模糊輪廓 |

## 使用時機

- **環境描寫**：新場景首次出現時
- **情緒高潮**：需要增強沉浸感時
- **潤色階段**：初稿完成後的強化

## 與其他 Skill 的關聯

- **協作**：`skill_scene_writer`（提供感官素材）
- **受影響於**：`skill_pacing_calculator`（決定詳細程度）

## 注意事項

1. **不是越多越好**：過多感官描寫會拖慢節奏
2. **服務於敘事**：每個感官細節都應該有意義
3. **避免陳詞濫調**：「鮮血的腥味」用太多就失去效果
4. **角色限制**：角色不會注意到自己已經習慣的氣味
5. **平衡分配**：不要總是只有視覺描寫
